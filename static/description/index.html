Autorisez les paiements avec Payzen

Description

Ce module implémente les paiements via le prestataire de paiement Payzen. Il permet :

  • De payer via Payzen

Installation

Lors de l'installation, ce module va ajouter un nouveau mode de paiement accessible lors du réglement des commandes et factures.
La configuration du module se fait par le menu Comptabilité > Paiements > Intérmédiares de paiement > Payzen.
Les valeurs à modifier sont :

  • Identifiant de la boutique
  • Certificat de test (pour les essais)
  • Certificat de production (pour la production)
Ces valeurs sont à récupérer sur le backoffice de Payzen.

Usage

Lors du paiement d'une commande, Payzen sera proposé comme moyen de paiement.
